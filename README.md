# ABBYY FineReader Online CLI.

- - -
**Usage of this program intended exceptionally for acquainting purposes**
- - -
#### Supported export formats:
- Docx
- Pdf
- Pdfa 
- Epub
- Xlsx
- Text

#### Install:
`python3 setup.py install`

#### Usage example:
`fineocr "Eng-Uk dict.pdf" -t pdf -l English+Ukrainian`
